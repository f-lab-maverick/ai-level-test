# F-Lab AI 서비스 개발자 역량 진단 테스트 v8.5.1

## 프로젝트 개요

F-Lab의 예비 멘티들을 대상으로 한 AI 활용 역량 진단 테스트입니다. 개발자들이 AI 러닝 갭을 스스로 인지하고, 자연스럽게 AI 학습 필요성을 느끼도록 설계되었습니다.

### 핵심 특징
- **25문항 완전판**: 실무 중심 핵심 개념 문제
- **6단계 레벨 체계**: Lv1 ~ Lv6 세분화된 역량 측정
- **실시간 결과 분석**: 레벨별 상세 피드백 제공
- **개인화 결과**: 레벨별 맞춤 학습 경로 제공

---

## 레벨 체계

### 6단계 레벨 구조

| 레벨 | 명칭 | 정답 수 | 정답률 | 핵심 역량 |
|------|------|---------|--------|-----------|
| **Lv6** | AI 아키텍트 | 24~25개 | 96%+ | 멀티모달 RAG, 안전 시스템 설계 |
| **Lv5** | AI 에이전트 엔지니어 | 21~23개 | 84%+ | Agent 설계, 메모리 구조 |
| **Lv4** | AI 시스템 설계자 | 17~20개 | 68%+ | Function Calling, 가드레일 |
| **Lv3** | AI 데이터 통합자 | 13~16개 | 52%+ | RAG, 청킹, 벡터 검색 |
| **Lv2** | AI 기능 구현자 | 8~12개 | 32%+ | LLM 개념, 프롬프트 기법 |
| **Lv1** | AI 활용자 | 0~7개 | ~32% | 기본 개념, AI 활용 입문 |

### 레벨별 상세 설명

1. **Lv1 - AI 활용자**
   - LLM 기본 이해, 프롬프트 엔지니어링 초급
   - ChatGPT/Claude 등 AI 도구 기본 사용

2. **Lv2 - AI 기능 구현자**
   - LLM 동작 원리, Temperature, Hallucination 이해
   - Few-shot, CoT 등 프롬프트 기법

3. **Lv3 - AI 데이터 통합자**
   - RAG, Embedding, VectorDB 설계
   - 청킹 전략, 하이브리드 검색

4. **Lv4 - AI 시스템 설계자**
   - Function Calling, 가드레일 설계
   - Structured Output, 에러 회복

5. **Lv5 - AI 에이전트 엔지니어**
   - AI Agent의 ReAct 패턴, 자기 복구
   - 멀티모달 RAG, 장기 메모리 설계

6. **Lv6 - AI 아키텍트**
   - 멀티모달 인덱싱 전략
   - 안전/컴플라이언스 시스템 설계

## 문항 구성

### 레벨별 문항 분포 (총 25문항)

| 레벨 | 문항 번호 | 문항 수 | 주요 토픽 |
|------|-----------|---------|-----------|
| Lv1 | Q1, Q2, Q13 | 3개 | LLM 한계, 프롬프트 기본, 비용 제어 |
| Lv2 | Q3, Q4, Q5, Q14 | 4개 | Hallucination, Temperature, 프롬프트 기법 |
| Lv3 | Q6, Q7, Q15, Q20, Q21, Q22 | 6개 | RAG, 청킹, 캐시, 하이브리드 검색, ANN |
| Lv4 | Q8, Q9, Q10, Q16, Q23, Q24, Q25 | 7개 | 가드레일, Function Calling, 리랭킹 |
| Lv5 | Q11, Q12, Q17 | 3개 | Agent 복구, 메모리, 멀티모달 RAG |
| Lv6 | Q18, Q19 | 2개 | 멀티모달 인덱싱, NSFW 검열 |

- **Lv3**: 6문항 중 3개 이상 정답
- **Lv4**: 7문항 중 4개 이상 정답
- **Lv5**: 3문항 중 2개 이상 정답
- **Lv6**: 2문항 중 1개 이상 정답

## 사용 방법

1. `index.html`을 브라우저에서 열기
2. "테스트 시작하기" 클릭
3. 25문항 순차 응답
4. 결과 페이지에서 레벨 및 상세 분석 확인
