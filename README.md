# AI 레벨테스트

## 링크

- 테스트: https://f-lab-maverick.github.io/ai-level-test
- 관리자: https://f-lab-maverick.github.io/ai-level-test/admin

- `f-lab-ai-test-questions.json` - 문항 데이터
